🚀 Compiler Design Lab – Final Boss IDE
📌 Project Overview

This project is a complete Compiler Design Laboratory Implementation built with an interactive Streamlit-based IDE interface. It integrates multiple phases of a compiler along with visualizations to enhance understanding and usability.

The system provides implementations for all 15 compiler lab experiments, combined into a single, user-friendly platform.

🎯 Objectives

Implement core concepts of Compiler Design
Provide interactive visualization of automata and DAGs
Build a mini compiler IDE for educational purposes
Simplify complex algorithms for better understanding

🧠 Features

💻 Interactive Streamlit UI
🔍 Lexical Analyzer (Tokenization)
🔄 Infix to Postfix Conversion
🌐 NFA Visualization (Graphviz)
🌳 DAG Visualization
📊 FIRST & FOLLOW Computation
⚙️ Modular structure for all experiments

🧪 List of Experiments
